# CharmingReads

- 유저가 작성한 독서평을 기반하여 책을 검색하는 사이트입니다

## 사용 기술

- 프론트 앤드
  - react, redux, redux-saga, styled-components
  - Next.js, @next/bundle-analyzer
  - axios, immer
- 백앤드
  - Node.js, express, cors
  - mysql, sequelize
  - passport, express-session
- 기타
  - AWS S3
